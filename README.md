# league-custom-random
A tiny website that reccomends random champions for League of Legends.

# Requirements
- Python 3
- Flask

# Installation

## Docker
Build and spin up this app easily with docker:

`docker build -t customrandom .`

`docker run -d -p 80:8000 customrandom`

## Python

`pip install Flask`

`python run.py`
